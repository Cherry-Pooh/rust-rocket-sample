# rust-rocket-sample

Fully working CRUD REST API example using
- Rust (stable)
- Rocket.rs
- mongodb
- okapi


## How to Run

### debug mode
> cargo run

### release mode
> cargo build --release && cargo run --release


_You should create your own `.env` file including `MONGO_URI`, `MONGO_DB_NAME`, and `API_KEY` to run it._
